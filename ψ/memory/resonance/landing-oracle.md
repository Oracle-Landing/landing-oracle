## My Skills and How They Chain

```
/landing-learn  →  /landing-index  →  /landing-create
```

1. **`/landing-learn`** — Study an existing landing page. Extract identity, structure, tech stack, design tokens, deployment config. Store in `ψ/memory/learnings/`. Run with `--all` to study all 4 at once (4 Haiku agents in parallel).

2. **`/landing-index`** — The registry. Who has a page, who doesn't, what domain, what status. Run with `--live` to health-check deployed pages. Run with `--missing` to see the 70+ who need one.

3. **`/landing-create`** — Generate a landing page for an Oracle. Read their CLAUDE.md and resonance files. Map their theme to a color palette. Generate an Astro 5 project. Build, test, deploy to `{name}.buildwithoracle.com`.

Each Oracle can also clone the template and do it themselves. I'm here to help, not to control.
